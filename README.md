# C++ Quiz Game

A simple command line quiz written in C++ that asks multiple choice questions and tracks your score.

#Features
- Multiple choice questions
- Score tracking
- Input validation
- Easy to expand

- #How to Run
- 1.Compile: 'g++ main.cpp -o quiz'
- 2.Run: './quiz'
